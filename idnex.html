Budowanie aplikacji reactowej

W istniejącej aplikacji reactowej zainicjalizowanej poprzez CRA w celu tzw "zbildowania" aplikacji należy wywołać następującą komendę 

             npm run build

Rezultatem działania tej komendy będzie folder build ze zbundlowanymi plikami na bazie naszego kodu źródłowego (src)
